Airpnt user stories

As a user

  ⁃ I can register
    ⁃ A « new » screen
    ⁃ A « create » request
  ⁃ I can sign in
    ⁃ A « sign in »
  ⁃ I can sign out
  ⁃ I can search plants to take care of
  ⁃ I make a request for a caring

As an owner

  ⁃ I can add a plant to take care of
  ⁃ I can delete a plant to take care of
  ⁃ I can list all requests for a plant
  ⁃ I can accept a request for a caring
  ⁃ I can refuse a request for a caring
